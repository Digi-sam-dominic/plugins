# @cap/decrypt-bundle

Native plugin to securely decrypt encrypted exam bundles and return local file paths for rendering.

## Install

To use npm

```bash
npm install @cap/decrypt-bundle
````

To use yarn

```bash
yarn add @cap/decrypt-bundle
```

Sync native files

```bash
npx cap sync
```
